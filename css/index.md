## Flip (mirror) elements

### Horizontally

```css
.element {
  display: inline-block; /* or display: block; */
  transform: scaleX(-1);
}
```

### Vertically

```css
.element {
  display: inline-block; /* or display: block; */
  transform: scaleX(-1);
}
```

### Horizontally and vertically

```css
.element {
  display: inline-block; /* or display: block; */
  transform: scale(-1, -1);
}
```
